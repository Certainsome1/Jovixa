# Project Rules

You are building an AI-powered engineering job matching platform.

Tech stack:
- Frontend: Next.js + Tailwind
- Backend: FastAPI
- Database: PostgreSQL
- AI/NLP: Python
- Vector DB: pgvector

Coding standards:
- Clean architecture
- Modular code
- Production-ready APIs
- Typed interfaces
- Reusable components
- Environment variables for secrets
- Docker-ready setup

Important:
- Prioritize MVP speed
- Keep files modular
- Use semantic naming
- Add comments only where necessary
- Avoid overengineering

Main goal:
Build a system that parses resumes, fetches fresh jobs, scores jobs semantically, and ranks best matches.
